Listas Ordenadas

Esse tipo de lista é sequencial podendo escolher tanto letras, números ou algarimos romanos.

  1. blabla
  2. blabla
  3. blabla
  4. blabla
  5. blabla

Listas Não Ordenadas

        Esse tipo de lista não tem sequência e por conta disso serve mais
            para fazer uma lista de marcações.

  • Blabla
  • Blabla
  • Blabla
  • Blabla
  • Blabla

Listas dentro de listas

Podemos colocar uma lista dentro de outra lista ordinaria ou não

  1. Blabla
    • BlaBla
    • BlaBla
  2. Blabla
    1. Blabla
